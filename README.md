# IFCT39 - Posicionamiento web y marketing digital en buscadores

Material del curso **IFCT39** del SEPE ([programa formativo oficial](https://sede.sepe.gob.es/es/portaltrabaja/resources/pdf/especialidades/IFCT39.pdf)): SEO, SEM, WordPress, analítica y marketing digital.

> Desde entender qué es internet a publicar un sitio propio, posicionarlo en Google y medirlo con GA4.
